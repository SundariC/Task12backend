# Task12 Backend

## Overview
This is the backend service for Task12, built with Node.js/Express (or your framework).

## Getting Started

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Installation
```bash
npm install
```

### Running the Server
```bash
npm start
```

The server will run on `http://localhost:3000` (adjust port as needed).

## API Documentation

### Using Postman
1. Import the API collection into Postman
2. Set environment variables (base URL, auth tokens, etc.)
3. Test endpoints using the provided requests
